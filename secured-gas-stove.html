Secured Gas Stove - Safety Monitoring System

Overview

Secured Gas Stove is an embedded safety system designed to detect gas leakage and automatically cut off gas supply to prevent fire hazards.

Objective

To improve household kitchen safety by implementing an automatic gas leak detection and control mechanism.

Key Features

  • Gas leakage detection using gas sensor
  • Automatic gas valve shutoff
  • Alarm system for user alert
  • Real-time monitoring

Technologies Used

  • Microcontroller
  • Gas Sensor (MQ series)
  • Embedded C / Arduino
