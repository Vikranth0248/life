Accessible Chessboard for Visually Impaired

Overview

This project is a Python-based chessboard application built using Pygame, specially designed for visually impaired users.

Features

  • Voice input for moves
  • Audio feedback
  • Keyboard navigation
  • Accessible interface design

Technologies Used

Python, Pygame, Speech Recognition

Challenges

Handling real-time voice input and ensuring smooth board interaction.
